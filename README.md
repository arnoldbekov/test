## HistoryTravel

**HistoryTravel** — это веб‑сайт с интерактивной картой культурного и исторического наследия Приднестровья.

На сайте доступны:
- **Карта** с отмеченными объектами наследия;
- **Исторические эпохи** с кратким описанием периодов;
- **Галерея** с фотографиями «тогда и сейчас»;
- **Маршруты** (готовые и пользовательские);
- **Избранное** для сохранения интересных мест и маршрутов;
- страница **«О проекте»** с описанием сервиса и инструкцией.

Сайт адаптирован под телефоны, планшеты и десктопы и развёрнут на GitHub Pages.
